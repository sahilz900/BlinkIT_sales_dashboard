# Blinkit Sales Analysis Dashboard (Power BI)

## 📊 Project Overview

This project presents an **interactive Power BI dashboard analyzing Blinkit sales data**.  
The dashboard provides insights into product performance, outlet characteristics, and sales distribution across different outlet types, sizes, and locations.

The goal of this project is to demonstrate **data visualization, business intelligence, and analytical insights using Power BI**.

---

## 🚀 Key Metrics

The dashboard highlights the following important KPIs:

- **Total Sales:** $1.2M  
- **Average Sales:** $141  
- **Number of Items:** 8,523  
- **Average Rating:** 3.9  

These KPIs help quickly understand overall business performance.

---

## 📈 Dashboard Insights

### 1️⃣ Fat Content Analysis

Sales are segmented by fat content:

- **Low Fat:** $776.3K  
- **Regular:** $425.4K  

This helps identify consumer preference for healthier product options.

---

### 2️⃣ Item Type Performance

The dashboard analyzes total sales across multiple product categories including:

- Dairy

- Baking Goods

Top performing categories include **Fruits & Vegetables and Snack Foods**.

### 3️⃣ Outlet Establishment Trends

This visualization shows how sales vary by **outlet establishment year**, highlighting the growth and expansion of retail outlets over time.

---

### 4️⃣ Sales by Outlet Size

Sales are compared across outlet sizes:

| Outlet Size | Total Sales |
|-------------|-------------|
| Medium | $507.9K |
| Small | $444.8K |
| High | $249.0K |

Medium-sized outlets generate the highest sales.

---

### 5️⃣ Sales by Outlet Location Tier

Outlets are grouped by location tier:

| Tier 3 | $472.13K |

| Tier 1 | $336.40K |

Tier 3 locations generate the most revenue.
